IIIF Hosting

IIIF Hosting - software as a service

The quickest way to get up and going is to upload images to an image hosting service. One quick one to get started with is IIIF Hosting.

  1. Click "Sign in/up"

  2. Click "Import"

  3. Drag or upload an image from your computer.

  4. Success, now you should be able to see a zoomable version of your image.

Now you have a hosted version of your image, using a IIIF Image API server. If you click the "info.json" link in the upper right-hand corner, you will see the Image Information response.
